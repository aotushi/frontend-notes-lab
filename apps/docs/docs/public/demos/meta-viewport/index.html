meta viewport 真实读数验证

这个页面不再模拟视口。上方读取当前浏览器真实 API 值；下方用一个真实 DOM 容器验证：`meta viewport` 只决定移动端布局基准，不会自动修好写死宽度的 CSS。

当前浏览器视口

layout viewport
window inner size
visual viewport
visual scale
DPR
document overflow

布局视口和视觉视口是浏览器坐标空间，不是源码里的块元素。它们只能通过浏览器 API 读取，或者用 DevTools 设备模式改变后观察。

对应 API

document.documentElement.clientWidth
window.innerWidth
window.visualViewport?.width
window.devicePixelRatio

固定宽度溢出实验

这个 DOM 块真实存在，源码里能找到：.fixed-box { width: 680px; }。当容器小于 680px，它会产生横向溢出。
