# 算法

算法分类收录面试高频的算法题型与解题套路。

## 已收录知识点

### 动态规划

- [斐波那契与动态规划](/algorithm/dynamic-programming/fibonacci-and-dp)：朴素递归的 O(2^n) 缺陷、状态转移方程 `dp[i] = dp[i-1] + dp[i-2]`、空间优化，及常见 DP 转移方程速查。

### 数组与查找

- [二分查找与洗牌算法](/algorithm/array-and-search/binary-search-and-shuffle)：有序数组二分查找 O(log n)、Fisher-Yates 洗牌 O(n) 与常见错误写法。
