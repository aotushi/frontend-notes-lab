# 前端资料源

这个文件是资料池，不直接作为最终学习结论。正式展示页同步到 `docs/resources/index.md`。

## 维护原则

- 优先收录官方文档、标准规范、长期维护的开源项目和可验证数据源。
- 单篇文章只在观点独特、可核验、仍然适用于当前前端生态时收录。
- 面试题库只作为查漏补缺材料，不作为知识结论来源。
- 旧题库和不再维护的资料不直接作为结论来源，但可以作为离线题库素材池保留。
- 资源失效、广告/搬运痕迹明显、无法定位作者或原始出处时删除。
- 从旧题库迁移题目时只保留仍适用于当前前端生态的问题；例如 jQuery 时代的兼容性技巧、旧版 Vue/React API 细节不进入正式文档，除非主题本身就是历史对比。
